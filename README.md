# Triftly www

Marketing site for [Triftly](https://www.triftly.app) — Explore · Plan · Spend · Share.

## Stack

Same family as `tgt-www`:

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**

## Live

- https://www.triftly.app/
- https://yky32.github.io/triftly-www/

## Local

```bash
npm install
npm run dev
```
